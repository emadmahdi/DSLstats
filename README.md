DSLstats is a tool to monitor ADSL & VDSL lines in routers. This repository is for a new version of DSLstats v6.5.10. This version fixes a bug that cause recording of the samples to fail with an error dialog "Invalid integer". I took the source code of this tool v6.5.9 from the original delveloper website at: https://dslstats.42web.io and used his instructions to rebuild the project.
